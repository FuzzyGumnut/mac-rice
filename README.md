## Installation

Clone the repository:

```bash
git clone https://github.com/FuzzyGumnut/mac-rice.git

cd mac-rice

chmod +x install.sh

./install.sh
```

---

## What Gets Installed

The installer will automatically install:

* AeroSpace
* SketchyBar
* Starship
* Fastfetch
* Borders

It will also copy all included configuration files into the correct locations.

---

## Required After Installation

### Install JetBrainsMono Nerd Font

Download and install:

https://www.nerdfonts.com/font-downloads

Search for:

```text
JetBrainsMono Nerd Font
```

---

### Enable macOS Permissions

Open:

```text
System Settings → Privacy & Security → Accessibility
```

Enable:

* AeroSpace
* SketchyBar

You may need to log out and back in after granting permissions.

---

## Optional Applications

Install these for full functionality:

* Spotify (Music Widget)
* Tailscale (Tailscale Status Widget)
* X-VPN (VPN Status Widget)
* Raycast

---

## Updating

To update to the latest version:

```bash
cd ~/mac-rice

git pull

./install.sh
```
